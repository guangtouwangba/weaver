# 白板应用核心操作术语表

本文档精简了白板应用中的核心操作名词与业务对象说明，便于快速查阅。

## 1. 引导与协作 (Facilitation & Collaboration)

| 操作名词 (Term) | 说明 (Description) |
| :--- | :--- |
| **Summon (召唤)** | 强制将所有参与者的视口（Viewport）瞬间跳转到引导者当前的位置和缩放级别。 |
| **Follow Me (跟随模式)** | 开启后，参与者的视图会实时、持续地同步引导者的平移和缩放操作。 |
| **Private Mode (隐私模式)** | 开启时，用户输入的内容（如便签文字）对他人隐藏；直到引导者执行“揭示”操作。 |
| **Reveal (揭示)** | 结束隐私模式，将所有隐藏的内容瞬间向所有参与者展示。 |
| **Voting (投票)** | 分配特定票数给参与者，使其能对画布上的对象进行点选投票，系统自动统计结果。 |
| **Timer (计时器)** | 共享的倒计时工具，用于控制会议节奏（Time-boxing），通常全员可见。 |
| **Hide Cursors (隐藏光标)** | 隐藏其他参与者的鼠标轨迹和名称标签，常用于减少干扰或保护匿名性。 |

## 2. 对象与编辑 (Objects & Editing)

| 名词 (Term) | 说明 (Description) |
| :--- | :--- |
| **Sticky Note (便签)** | 基础信息单元，模拟物理便签，通常支持文本自动缩放和颜色编码。 |
| **Frame (画板/容器)** | 特殊容器对象。不仅用于分组，还常作为演示文稿的“幻灯片”页面或导出区域。 |
| **Connector (连接线)** | 连接两个对象的线条。具备**正交路由**能力，能自动避开遮挡物。 |
| **Anchor Point (锚点)** | 对象边缘或中心的特定点，连接线可吸附于此。 |
| **Binding (绑定)** | 连接线与对象之间的吸附关系。对象移动时，绑定的线条会自动重绘跟随。 |
| **Group (编组)** | 将多个对象逻辑上组合在一起，便于统一移动或缩放，解组后关系消失。 |
| **Lock (锁定)** | 禁止选中或移动特定对象，防止误操作（通常用于背景图或模板框架）。 |

## 3. 视图与导航 (View & Navigation)

| 名词 (Term) | 说明 (Description) |
| :--- | :--- |
| **Infinite Canvas (无限画布)** | 无边界的二维操作空间，支持自由的平移（Pan）和缩放（Zoom）。 |
| **Viewport (视口)** | 用户当前屏幕可见的画布区域。 |
| **Mini-map (小地图)** | 位于角落的全局导航器，显示画布全貌及当前视口位置，点击可快速跳转。 |
| **Grid Layout (网格布局)** | 辅助对齐系统，帮助便签或对象自动吸附到整齐的行列中。 |

## 4. 业务流程 (Workflows)

| 名词 (Term) | 说明 (Description) |
| :--- | :--- |
| **Kanban (看板)** | 具备容器属性的列布局。便签拖入不同列时，会自动吸附并可能触发状态变更。 |
| **Clustering (聚类)** | AI 辅助功能，根据颜色或文本语义，自动将散乱的便签分类整理。 |
| **Dot Voting (点数投票)** | 模拟线下工作坊，通过放置圆点（Dot）来表达优先级的决策方式。 |
